# KonX Affiliate Dashboard

A custom WordPress/WooCommerce plugin that provides a complete affiliate management system.

## Requirements

- WordPress 5.8+
- PHP 7.4+
- WooCommerce 6.0+

## Project Structure

```
konx-affiliate-dashboard/
├── docs/                 # Project documentation
├── admin/views/          # Admin-facing view templates
├── assets/css/           # Stylesheets
├── assets/js/            # JavaScript files
├── includes/             # Core plugin classes and functions
├── public/views/         # Public-facing view templates
├── templates/            # Overridable templates
├── languages/            # Translation files
├── konx-affiliate-dashboard.php  # Plugin bootstrap
├── uninstall.php         # Cleanup on uninstall
└── readme.txt            # WordPress plugin readme
```

## Installation

1. Clone this repository into your `wp-content/plugins/` directory.
2. Activate the plugin through the WordPress admin.
3. Ensure WooCommerce is installed and activated.
